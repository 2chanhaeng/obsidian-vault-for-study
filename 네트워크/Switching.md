[[LAN]] 에서 [[MAC Address|MAC 주소]]를 기반으로 데이터를 특정 목적지로 보내기 위해 경로를 동적으로 결정하는 기술

1. 수신한 [[MAC Address|MAC 주소]] 확인
2. 스위치 테이블을 참조하여 MAC 주소 포트 확인
3. 주소가 테이블에 있는 경우 [[Unicast|해당 포트로 데이터 전송]]
4. 없는 경우 [[Broadcast|전 포트로 데이터 전송]]
